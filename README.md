# Actividad 3: Desarrollo Full Stack

# Gestor de Tareas - Express

Este proyecto es una aplicación backend desarrollada con Node.js y Express que se ejecuta en entorno local (localhost). Permite autenticación de usuarios y gestión de tareas.

---

## Requisitos

Antes de ejecutar el proyecto, asegúrese de tener instalado:

- Node.js
- npm


## Ejecución del proyecto en localhost

### 1. Abrir Command Prompt

En Windows, abra **Command Prompt** (Símbolo del sistema).

---

### 2. Ubicarse en la carpeta del proyecto

Use el comando `cd` hasta llegar a la carpeta donde se encuentra el archivo `server.js` (o `index.js`):

```bash
cd ruta/del/proyecto
```
---
### 3. Instalar dependencias (solo la primera vez)

Ejecute el siguiente comando:

```bash
npm install
```

### 4. Iniciar el servidor

Ejecute el servidor con el siguiente comando:

```bash
node server.js
```

Si el servidor se inicia correctamente, aparecerá el mensaje:

```bash
Servidor Audi corriendo en http://localhost:3000
```

Acceso desde el navegador

Abra un navegador web y diríjase a la siguiente dirección:

```bash
http://localhost:3000
```

Esto cargará la página principal ubicada en la carpeta public.

---
### Credenciales de prueba

Para iniciar sesión desde la página web, utilice las siguientes credenciales:

```bash

```
